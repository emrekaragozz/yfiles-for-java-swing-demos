Step 4: Modifying the Visualizations

The visual appearance for each type of graph element (except bends) is specified by means of styles.

By default, all newly created elements get the style set to the NodeDefaults and EdgeDefaults properties of IGraph.

To set a specific style for a graph element, either create it with one of the various createNode/createEdge/addLabel/addPort methods that have a style parameter or change the style property later on.

yFiles already comes with a number of useful styles for most graph element types. The custom style tutorial shows how to create your own custom style for nodes, edges, ports and labels.

See the sources for details.
