Tutorial 2: Custom Styles

This tutorial is a step-by-step guide to customizing the visual representation of graph elements. It shows the creation of custom styles for nodes, edges, labels, and ports. Moreover, it presents a custom arrowhead rendering, a customized edge selection, and a visual representation of graph elements that depends on the current application state. Finally, several optimization strategies are discussed.

It is intended for users who want to learn how to create custom styles from scratch. If you are new to styles, in particular to their customization, we recommend to start by going through the tutorial steps that show the basics one by one. Of the specialized topics, you can skip the ones that doesn't apply to your use case. To make full use of the tutorial, we recommend to review and possibly modify the source code of the sample projects.

Running a Tutorial Step

The best way to run a tutorial step is from within your preferred IDE. Set the directory <yFiles-for-Java>/tutorials/ as source directory of your project and add the yFiles library (yfiles-for-java-swing.jar) to your references. Each step can be started by running or debugging its class SampleApplication.

Steps in this Tutorial

Name Description
Step 01: Custom Node Style Shows how to create a custom node style.
Step 02: Node Color Shows how to change a style dependent on the node's Tag property.
Step 03: UpdateVisual Shows how to implement high-performance rendering of nodes.
Step 04: Improve Rendering Performance Shows how to improve the rendering performance of nodes using pre-rendered images.
Step 05: Drop Shadow Shows how to draw a well-performing drop shadow.
Step 06: IsInside Shows what to do to make edges correctly cropped at the node's visual bounds and to determine whether a location, e.g. of a port, is inside the visual bounds or not.
Step 07: Hit Test Shows how to implement a proper hit detection for mouse clicks and marquee selection.
Step 08: GetBounds Shows how to include the shadow of a node in its bounds calculation. This is used for example when the viewport is adjusted to show the complete graph.
Step 09: Line from Node to Label Shows how to visually connect a label to its owner node with a line.
Step 10: IsVisible Shows how to keep a connector of a label visible as long as it is within the viewport, even if the owner is currently outside of the viewport.
Step 11: Custom Label Style Shows a custom label style.
Step 12: Label Preferred Size Shows how to set the size of the label depending on the size of its text.
Step 13: High Performance Rendering of Labels Shows how to implement high-performance rendering for labels.
Step 14: Label Edit Button Shows how to display a button on a label that starts the label editor.
Step 15: Button Visibility Shows how to hide the 'Label Edit' button dependent on the zoom level.
Step 16: Using Data In Label Tag Shows how to use data from a business object, which is stored in the label's tag, for rendering.
Step 17: Custom Edge Style Shows how to create a custom edge style which allows to specify the edge thickness by setting a property on the style.
Step 18: Edge Hit Test Shows how to take the thickness of the edge into account when checking if the edge was clicked.
Step 19: Edge Cropping Shows how to crop an edge at the node bounds.
Step 20: Animated Edge Selection Shows how to change the style of an edge if the edge is selected.
Step 21: Adding Arrows Shows how to add arrows to the edges.
Step 22: Custom Arrow Shows a custom arrow.
Step 23: Arrow Thickness Shows how to render the arrow dependent on a property of the edge it belongs to.
Step 24: Custom Ports Shows a custom port style.
Step 25: Style Decorator Shows how to implement a node style decorator that enhances an existing style by adding visual decorators.
Step 26: Custom Group Style Shows how to implement a special node style for group nodes.
Step 27: Custom Group Bounds Shows how to customize the calculation of the group insets.
Step 28: Bridge Support Shows how to enable bridges for a custom edge style.
