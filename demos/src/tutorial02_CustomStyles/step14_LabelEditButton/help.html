Step 14: Label Edit Button

Implement a button within a label to open the label editor. The button is defined in class MySimpleLabelStyle so it is available in all labels.

Things to Try

  • Click the edit button to open the label editor.
  • Take a look at LabelVisual.update() and LabelVisual.paint() as well as class EditActionButtonProvider.
