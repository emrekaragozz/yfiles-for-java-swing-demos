Step 10: Grouping

Grouping adds hierarchical structure to a graph by assigning one or several nodes to a common group node. Note that collapsing and expanding of groups is introduced in the next step.

GraphEditorInputMode provides the following default gestures for grouping:

  • Group the currently selected nodes with Ctrl/Command+G.
  • Ungroup the currently selected nodes with Ctrl/Command+U. Note that this does neither automatically shrink the group node nor remove empty groups.
  • Shrink a group node to its minimum size with Ctrl/Command+Shift+G.
  • Press Shift when dragging nodes into or out of groups to change the graph hierarchy.

See the sources for details.
