Description

This demo shows how to use split IDs to align edges outside group nodes with edges inside group nodes when arranging graphs with RecursiveGroupLayout.

With this algorithm, the routing for inter-edges (i.e. edges connecting nodes outside a group with nodes inside said group) is sometimes not satisfactory because these edges are routed after the rest of the graph is already arranged. Thus inter-edges sometimes do not fit into the overall layout style.
However, when replacing a given inter-edge with two separate edges - one connecting the node outside to the group node and one connecting the group node to the node inside - and using HierarchicLayout as the core layout algorithm, RecursiveGroupLayout's support for split edges and HierarchicLayout's support for direct group content edge routing may be used to align the two split edges at the corresponding group node's border.
