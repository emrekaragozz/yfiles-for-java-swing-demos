Description

Customize the reshape handles of nodes by implementing the IReshapeHandleProvider interface and adding instances to the node decorator of the graph.

Things to try

  • Select and resize the nodes.

Node Types

  • Red nodes cannot be resized. They don't display resize handles.
  • Green nodes show only four handles at the corners. During resizing, these handles maintain the aspect ratio of the node.
  • Orange nodes cannot extend beyond the black rectangle. They can only be resized so that the new bounding box stays withing the enclosing black rectangle.
  • Blue nodes combine the behavior of the orange and green nodes.
