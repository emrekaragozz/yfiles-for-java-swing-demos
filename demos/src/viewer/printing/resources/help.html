Description

By means of the CanvasPrintable class yFiles for Java (Swing) provides an API to print the contents of a CanvasComponent.

Things to try

  • Switch between the tabs Graph and Print Preview to edit the graph or preview the outcome and adjust the printing settings.
  • The red dotted rectangle specifies the area that will be printed. To print the whole viewport, disable the setting Clip Printable Area.
  • Press Page Format on the Print Preview tab to show settings for the page size, margins and orientation or press Print to open the print dialog.
