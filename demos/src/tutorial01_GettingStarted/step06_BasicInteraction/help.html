Step 6: Basic Interaction

The GraphEditorInputMode is an input mode that supports a large number of ways to interact with the graph and its view, for example moving, deleting, creating, and resizing of graph elements.

To select a single element just click it with the mouse. Press Alt to step through the different possible hits (e.g. to select a node label inside its owner). To select multiple elements, either extend an existing selection by pressing Ctrl while clicking, or drag a selection rectangle over all graph elements that you want in your selection. Ctrl/Command-A selects all elements.

Resize nodes with the handles of selected nodes.

To move a node or bend, just drag it when it is selected.

To create an edge, start dragging anywhere on the unselected source node and stop dragging on the target node.

To create an edge bend, click and drag the edge at the desired bend location. To create a bend during edge creation, release the left mouse button anywhere on an empty spot the canvas while dragging. Then click to create more bends.

Nodes may provide multiple ports. You can create an edge directly between specific ports and move the source or target to a different port at any time. Note that this is not part of this step but shown later.

To create or edit a label, just press F2 when the owner is selected.

To move a label, just drag it to the desired location. If the label model supports only discrete positions, these positions are indicated by empty rectangles when you start dragging the label. In this case, you can only move a label to one of these positions.

See the sources for details.
