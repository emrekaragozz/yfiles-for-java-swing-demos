Description

Customize the movement behavior of nodes by implementing the IPositionHandler interface and adding instances to the node decorator of the graph.

Things to try

  • Select and move the nodes.

Node Types

  • Red nodes are unmovable.
  • Green nodes move only vertically or horizontally from the current start point.
  • Orange nodes move freely within the displayed rectangle but cannot leave it.
  • Blue nodes combine the green and orange restrictions: they move only vertically or horizontally and cannot leave the rectangle.
