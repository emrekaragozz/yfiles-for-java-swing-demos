Snapping

This demo shows how to enable snapping (guide lines) for graph elements. This is provided through class GraphSnapContext.

Snapping helps you to arrange graph items interactively without running an automatic layout by providing useful lines and position hints that elements snap to.

In addition to this, labels with certain label models can snap to particular positions relative to their owner, for example to the corners of a node or along the edge path. This can be enabled with class LabelSnapContext.

See the sources for details.
