Step 14: Automatic Graph Layout

Using layout algorithms can make your graph easier to read by automatically generating high-quality graph drawings.

An action for calculating an automatic layout is specified and added to the toolbar as a button.

The layout calculation can be started with the new toolbar button Layout Graph.

Since the graphs from the previous steps are not really suited for automatic layout, this step loads a different graph from a file.

See the sources for details.
