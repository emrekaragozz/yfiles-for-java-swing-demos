Description

Configure the GraphEditorInputMode so that only one graph item can be selected at a time.

Things to try

  • Toggle single selection mode using the respective button.
  • Try to drag the mouse over a set of nodes to select them. Marquee selection is completely disabled in single selection mode.
  • Click select a node. Try to extend the selection by holding the Ctrl/Command key down and click another node.
  • Click select a node. Try to extend the selection by holding the Shift key down and press one of the arrow keys to extend the selection to a certain direction.
  • Try to select all nodes by pressing Ctrl/Command-A.
  • Select a single node. You can toggle the selection state by hitting Ctrl/Command-Space. This works for single selection as well as in normal mode due to a custom command binding.
  • Click select a single node. Move the item focus (indicated by a thin dashed line) by holding the Ctrl/Command key down and pressing one of the arrow keys. Toggle the selection state of the focused item by pressing Ctrl/Command-Space.
  • Take a look at the source code, especially enableSingleSelection.
