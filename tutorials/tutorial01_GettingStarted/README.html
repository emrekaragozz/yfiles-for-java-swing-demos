yFiles for Java (Swing) Tutorials 1: Getting Started

Tutorial 1: Getting Started

This tutorial is a step-by-step introduction to the concepts and main features of the yFiles for Java (Swing) diagramming library. Topics include custom styles, full user interaction, Undo/Redo, clipboard, I/O, grouping and folding.

It is intended for developers who want to get started with the library in an effective way. If you are new to the yFiles for Java (Swing) diagramming library, we recommend to start by going through the projects in this tutorial one by one. To make full use of the tutorial, we also recommend to review and possibly modify the source code of each sample project.

Running a Tutorial Step

The best way to run a tutorial step is from within your preferred IDE. Set the directory <yFiles-for-Java>/tutorials/ as source directory of your project and add the yFiles library (yfiles-for-java-swing.jar) to your references. Each step can be started by running or debugging its class SampleApplication.

Steps in this Tutorial

Name Description
Step 01: Creating the View Introduces class GraphComponent, which is the central UI element for working with graphs.
Step 02: Creating Graph Elements Shows how to create the basic graph elements.
Step 03: Managing the Viewport Shows how to work with the viewport.
Step 04: Modifying the Visualizations Shows how to configure the visual appearance of graph elements using styles.
Step 05: Placing Labels Shows how to control label placement with the help of so called label model parameters.
Step 06: Basic Interaction Shows the default interaction gestures that are provided by class GraphEditorInputMode.
Step 07: Undo and Clipboard Shows how to use the undo and the clipboard features.
Step 08: Using GraphML for Loading and Saving Shows GraphML input and output.
Step 09: Customizing User Interaction Shows how to customize user interaction by decorating the graph with custom behavior.
Step 10: Grouping (Group Nodes) Shows how to configure the support for grouped (hierarchically organized) graphs.
Step 11: Folding Shows how to enable collapsing and expanding of group nodes.
Step 12: Binding Custom Data to Graph Elements Shows how to bind data to graph elements.
Step 13: Exporting Custom Data in GraphML Shows how to read and write data that is bound to graph elements to and from a GraphML file.
Step 14: Automatic Graph Layout Shows an outstanding feature of yFiles for Java (Swing): sophisticated layout algorithms that automatically generate high-quality graph drawings.
Step 15: Commands Shows how to use the built-in command framework of yFiles.
Step 16: Snapping Shows how to enable snapping (guide lines) for graph elements.
Step 17: Grid Snapping Shows how to enable grid snapping for graph elements.
Step 18: Orthogonal Edge Editing Shows how to enable interactive orthogonal edge creation.
