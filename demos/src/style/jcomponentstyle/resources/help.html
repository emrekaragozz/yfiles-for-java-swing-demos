Description

An outstanding feature of yFiles for Java (Swing) is the ability to use any Swing components directly as graph elements. Combine the layout and visualization capabilities of yFiles with your custom Swing components or Swing application!

Things to try

  • Select nodes or edge labels to see custom selection highlighting (borders) defined in the node and label components.
  • Change the focused node using Ctrl/Command + <arrow key> to see custom focus highlighting (background color of the top label) defined in the node components.
  • Create new customers by Shift clicking and new products by Ctrl/Command clicking on the background.
  • Create new relations between customers and products with the component label visualization. Note that edges can only be created from customer to product nodes.
  • Edit the name or id of a customer or product by clicking on the name or id and editing the text field. Note how the labels of connected edges are updates when the text fields lose focus after editing.
  • All edits of node properties are written back to the custom user-tag data of the nodes. Save a modified graph and have a look at the GraphML file to see the changed user-tag data.
