Grid Snapping

This demo shows how to enable grid snapping for graph elements. Also class GridVisualCreator provides the visualization of the grid.

  • Click the Show Grid button in the toolbar to toggle the grid's visibility. The snapping feature is unaffected.
  • Use the Grid Snapping box to choose how items snap to the grid.
  • Select and move a node or bend to make it snap to the grid.
  • Select and resize a node to make it snap to the grid.

See the sources for details.
