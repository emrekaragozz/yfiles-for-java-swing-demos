Quick Reference

Experience intuitive diagram creation, editing capabilities, and automatic diagram layout provided by yFiles for Java (Swing).

It is very easy to add, delete, move and label nodes and edges to create a nice diagram. You can interact with the diagram using zoom and pan, undo/redo, copy/paste, and much more.

Creating nodes

  • Click at an empty location on the canvas.

Creating edges

  • Drag the mouse with the left button held down on a node.
  • Complete the creation by releasing the mouse button on a target node.
  • Abort the creation by pressing the Esc key.
  • Create control points (bends) during edge creation by clicking on an empty area of the canvas, remove the last control point and finally cancel edge creation with Right-Click.

Creating and editing labels

  • Press the F2 key while an element or connection is selected.
  • Alternatively double-click an element, connection or label.

Panning and zooming

  • Move the diagram view by dragging the mouse while Ctrl/Command is pressed.
  • The zoom factor is changed with the mouse wheel or the zoom buttons in the toolbar.
  • Use Ctrl/Command + mouse wheel to zoom in or out on the center of the displayed area.
  • Display the entire diagram using the Fit Contents button in the tool bar.

Selection

  • Single object: click these items with the left mouse button. If the Ctrl/Command key is pressed, clicked items are added to the current selection.
  • Multiple objects: you can toggle between marquee and lasso selection in the toolbar. Drag the mouse over the canvas. All elements inside the resulting rectangle/lasso path will be selected.
  • All objects are selected by pressing Ctrl/Command + A.
  • Deselect single items: click on them with the Ctrl/Command key held down. Deselect all items: click on an empty part of the canvas.
  • Select labels: Click on the label while holding down the Shift key.

Moving items

  • Selected elements: drag the mouse, starting over a selected item.
  • Move selected labels: drag to the indicate positions.
  • Note that items have to be selected before they can be moved.

Resizing elements

  • Drag one of the handles that are shown when an element is selected.

Clipboard

  • Cut or copy selected elements to the clipboard by pressing Ctrl/Command + X or Ctrl/Command + C, respectively. The clipboard's contents can be pasted into the diagram with Ctrl/Command + V.
  • Cut, copy, and paste are also available in the tool bar and the Edit menu.

Undo/Redo

  • The last edit operations can be undone by pressing Ctrl/Command + Z, clicking the undo button in the toolbar or selecting Undo from the Edit menu. Undone operations can be restored by pressing Ctrl/Command + Y, the redo button or selecting Edit/Redo from the menu.

Removing or duplicating elements

  • All currently selected items are removed by pressing the delete key. Removing is also available in the tool bar and the Edit menu.
  • All currently selected items are duplicated by pressing Ctrl/Command + D. Duplicating is also available in the Edit menu.

Changing edge direction

  • To change the direction of the currently selected edges press Ctrl/Command + R or the appropriate menu item in the Edit menu.

Grouping

  • Selected nodes can be grouped into a group node by pressing Ctrl/Command + G. The group node's site will be adjusted to its contents. A node can be added to a group node or removed from a group node by dragging it with the Shift key held down. The group which will be the node's parent is indicated by highlighted corners.
  • Deleting a group node will remove the group node but not its children.

Collapsing and expanding groups

  • Collapse a selected and expanded group node by pressing Ctrl/Command + Minus. Collapse is also available in the Grouping menu. Collapse an unselected and expanded group node by clicking on the Minus button in the top left corner of the node.
  • Expand a selected and collapsed group node by pressing Ctrl/Command + Plus. Expand is also available in the Grouping menu. Expand an unselected and collapsed group node by clicking the Plus button in the top left corner of the node.

Entering and exiting groups

  • Enter a selected group node by pressing Ctrl/Command + ENTER. Enter is also available in the Grouping menu.
  • Exit a group node by pressing Ctrl/Command + BACKSPACE. Exit is also available in the Grouping menu.

Orthogonal edge creation

  • Switch orthogonal edges on/off in the toolbar.
  • Change the outgoing node side by pressing the space bar while creating an orthogonal edge (before creating a bend).

Snapping

  • Switch snapping on/off in the toolbar.
  • Drag nodes, edges and bends near other graph elements to make them snap.
  • Drag node and edge labels along the border/path of their owner to make them snap.
  • Switch on snapping to make orthogonal edge segments snap to other graph elements during edge creation.
