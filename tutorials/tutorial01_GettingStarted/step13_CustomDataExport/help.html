Step 13: Exporting Custom Data in GraphML

The I/O support can read and write custom data that is bound to graph elements to and from a GraphML file.

In this demo the creation time of each node is stored as LocalDateTime in an IMapper as described in the previous step. A tooltip shows the date for each node.

See the sources for details.
