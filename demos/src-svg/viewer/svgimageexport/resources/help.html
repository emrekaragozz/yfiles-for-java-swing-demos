Description

With the use of the Batik library yFiles for Java (Swing) supports exporting the content of a CanvasComponent in the SVG format. Furthermore, it is able to draw arbitrary objects and interact with them via InputModes.

Things to try

  • Resize and move the clipping rectangle to control the exported region.
  • See a preview of the exported image in the Preview tab.
  • Uncheck Show Decorations to hide the export of selection and clipping rectangle visualizations as well as other present adorners.
  • Uncheck Export Rectangle Contents to ignore the clipping rectangle and export the contents of the whole viewport.
  • Uncheck Transparent to use the canvas background color for the exported SVG.
  • Pan the exported image in the preview panel or use the mouse wheel for zooming.
