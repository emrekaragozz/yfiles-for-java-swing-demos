Step 11: Custom Label Style

Create a custom label style.

Things to Try

  • Take a look at class MySimpleLabelStyle.
  • Select a label and open the label editor using F2 to edit the label's text. The size of the label doesn't adjust to the size of the text. If you move the node label from North to East, for example, this results in an overlap.
  • Create a label using Shift+F2 when a node or an edge is selected.

Left to Do

  • Make the label size adjust to its text.
  • Override updateVisual() to get high-performance rendering.
  • Implement a button to open the label editor.
  • Allow to change the background color of labels.
  • Create a custom edge style.
  • Create a custom port style for nodes.
