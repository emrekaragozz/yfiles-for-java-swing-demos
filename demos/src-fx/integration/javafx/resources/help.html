JavaFX 8 introduces the SwingNode class, which enables embedding yFiles for Java (Swing) in a JavaFX application.

  • A toolbar that provides JavaFX buttons to change the zoom level of the canvas that is a Swing component as well as JavaFX buttons for undo/redo functionality.
  • A right click on a node shown on the canvas opens a JavaFX context menu and allows the user to delete the clicked node from the canvas.
  • On the left side a JavaFX palette offers nodes with different styles that can be dragged into the canvas.

Things to try

  • Change the zoom level by clicking on buttons in the toolbar.
  • Right click on a node to show a JavaFX context menu.
  • Drag a node from the palette and drop it on the canvas.
