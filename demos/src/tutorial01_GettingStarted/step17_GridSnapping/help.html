Step 17: Grid Snapping

Beyond snapping to other items yFiles also supports snapping of graph elements to a grid.

You can enable (or disable) interactive snapping of graph elements to grid points with the tool bar button Show grid. This also enables (or disables) the visualization of grid points.

The drop-down list in the tool bar provides these grid snapping policies

  • No grid snapping at all
  • Snap to horizontal grid lines
  • Snap to vertical grid lines
  • Snap to horizontal and vertical grid lines
  • Snap to grid points
  • Snap to horizontal and vertical grid lines as well as to grid points

See the sources for details.
