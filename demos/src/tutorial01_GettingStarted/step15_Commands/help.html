Step 15: Commands and Command Binding

The built-in command framework of yFiles can be used to easily connect actions with shortcuts that benefit from automatic management of enabling/disabling.

Note that using custom commands is purely optional: all yFiles functionality can be accessed with conventional GUI programming techniques, e.g. plain event listeners. Commands can however help to reduce code coupling and encapsulate user interaction code.

In this step we specify a command for calculating the automatic layout introduced in the previous step.

The layout calculation can now be started with the shortcut Ctrl/Command+L as well as with the toolbar button.

See the sources for details.
