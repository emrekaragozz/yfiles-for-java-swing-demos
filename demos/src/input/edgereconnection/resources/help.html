Description

Enable and customize the reconnection behavior of edges to other ports by implementing the IEdgeReconnectionPortCandidateProvider interface and adding instances to the edge decorator of the graph.

Things to Try

Select an edge and drag one of its port handles to another node. When over a node, green indicators highlight valid new locations for the port.

Edge Types

  • Red edges cannot be reconnected to another ports.
  • Orange edges can be reconnected to other orange nodes, but you cannot create self-loops.
  • Blue edges can be connected to all other ports. Note that the blue edge will reuse existing ports for other blue nodes and create new ports at the same location for other nodes.
  • Green edges connect to any location inside of green nodes when the Shift key is pressed.
