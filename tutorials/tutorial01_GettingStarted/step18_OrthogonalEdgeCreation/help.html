Step 18: Orthogonal Edge Editing

The orthogonal edge editing feature helps the user to interactively create and maintain orthogonal edge paths.

To create bends during edge creation, release the left mouse button anywhere on an empty spot of the canvas while dragging. Then, each additional click creates more bends and each new segment of the edge is either horizontal or vertical.

When a bend of an existing edge is added, deleted, or moved, the adjacent segments stay orthogonal.

See the sources for details.
