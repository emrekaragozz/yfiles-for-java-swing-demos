yGuard is a Java byte code obfuscator that can be used for effective name obfuscation of Java byte code. It is freely available at www.yworks.com/products/yguard.

Obfuscation

Obfuscation changes class, field, and method names in a consistent but nonsensical way that does not change the functionality of the software but makes decompiled Java code harder to read and publicly available API description of the original code inapplicable to the obfuscated code. Obfuscation is an important step in the process of deploying an application in order to protect your software. See how to obfuscate the yFiles library, which is required per license agreement for deploying an application that uses it.

Since some names need to be kept at runtime in order for Java applications to work, yGuard provides several mechanisms to control the obfuscation process. In this demo, annotations are used for this. For a fully documentation of the annotation mechanism click on the yGuard tab above or visit our website at www.yworks.com.
