Description

Explore the event system of yFiles.

The events are logged to the Event Log panel. Event logging can be switched on and off for different kinds of events using the Event Log Options. When looking for an event you want to subscribe to, you should generally start looking at the top of the list above and work your way down. I.e. in most cases it is recommended to use events of a particular input mode instead of subscribing to the low-level graph events (which should be reserved for rare, exceptional cases where the layers above won't suffice).

Things to try

  • Toggle the various event log options to see different kinds of events.
  • Create, delete, move, resize and edit graph items.
  • Move the whole graph within the viewport by dragging the mouse (Ctrl-Drag in Edit Mode).
  • Select and deselect graph items.
  • Expand, collapse, enter and exit group nodes.
  • Toggle the input mode using the "Toggle Editing" button.
