Step 9: Customizing Behavior

The ports that an edge can connect to interactively can be customized by means of the look-up mechanism.

To change an edge to a different port (on the same node), select the edge and drag its source or target handle to the new port. The valid port locations are highlighted when you start dragging. Note that in this demo edges can only be reassigned to ports at the same node.

See the sources for details.
