Step 16: Using Data In Label Tag

Adjust rendering depending on data from a business object stored in the label's tag. This mechanism also applies to nodes, edges and ports. In this example, a dummy business object, which only defines a color, is stored. This color is used to set the background of the label. You can store any object you like in the tag to use its data in the style.

Note that, in contrast to MySimpleNodeStyle, you can change the color of each label separately without affecting other labels that use the same style instance.

Things to Try

  • Change the label tags in method SampleApplication.createLabelTags() and see what happens.
  • Take a look at class MySimpleLabelStyle, especially method getColor().
