Description

Enable and customize the drag-and-drop support for nodes, labels and ports.

Things to try

Item Creation

  • To create a node, drag the desired node style from the left panel onto the canvas.
  • Or select a node style in the left panel and click on the canvas to create a node with the selected style.
  • Drop a label over a node or an edge to add it to it. Note that the Node Label can only be dropped over a node while the Edge Label can only be dropped over an edge.
  • Drop the port over a node to add it to that node.

Snapping

  • Drag the node around on the canvas to see grid snapping.
  • Drag a node around near another node to see node to node snapping.
  • Switch off snapping or snapping and preview using the combo box above the canvas.

Grouping

  • Drag a node over a green node to see group node highlighting.
  • Drop a node onto a green group node to place it inside the group.
  • Drop a node onto a normal green node to convert it to a group.
  • Red group nodes don't accept dropped nodes as their children.
  • Red normal nodes cannot be converted to group nodes.
