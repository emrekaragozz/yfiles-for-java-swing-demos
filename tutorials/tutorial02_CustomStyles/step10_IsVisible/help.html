Step 10: IsVisible

A connector to a label should be visible as long as it is within the view port, even if the node itself is currently outside of the view port. This behavior may be achieved by overriding the method isVisible() of class AbstractNodeStyle and have it check if any part of the node is within the view port. Otherwise the node is removed from the yFiles internal scene graph entirely.

Things to Try

  • Take a look at MySimpleNodeStyle.isVisible().
  • If you scroll the view so that the node with the label lies completely outside the view port, but the connector to the label is still inside, the connector remains visible. If you uncomment the first line of isVisible() so the base implementation is used and do the same thing, you will notice the connector disappears as soon as the node is outside of the view port.
