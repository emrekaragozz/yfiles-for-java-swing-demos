Step 2: Creating Graph Elements

The IGraph interface defines fundamental graph structure-related functionality including creation and deletion of graph elements.

The most important ones are nodes and edges. Nodes represent entities and are the only element type that can exist independent of other elements.

Connections between those entities are called edges. An edge can't exist without the two nodes it connects.

Control points of edges that split an edge into multiple segments are called bends.

Nodes can have ports that make it possible for edges to connect to nodes.

Finally, nodes, edges and ports can have labels which add textual information.

See the sources for details.
