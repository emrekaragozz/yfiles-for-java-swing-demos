Description

The InteractiveOrganicLayout provides organic layout for use in interactive environments.

A custom IPositionHandler automatically triggers layout updates when graph elements have been moved interactively.

Things to try

  • Create nodes by clicking on the canvas and interconnecting them by dragging from one (unselected) node to another. Select nodes and edges to delete them using the delete key.
  • Select nodes by clicking them or using marquee selection.
  • Move selected nodes around by dragging them around the screen. Note how an automatic organic layout algorithm is applied while you drag the node and after the drag operation is finished.
