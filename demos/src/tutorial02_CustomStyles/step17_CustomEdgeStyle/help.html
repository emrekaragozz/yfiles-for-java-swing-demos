Step 17: Custom Edge Style

Create a custom edge style which allows to specify the edge thickness by setting a property on the style. The createVisual and updateVisual methods are implemented like in class MySimpleNodeStyle and MySimpleLabelStyle.

Things to Try

  • Zoom in on an edge and try selecting it by clicking near its borders.
