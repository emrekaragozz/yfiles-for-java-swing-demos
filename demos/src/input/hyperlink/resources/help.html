Description

Add hyperlink support for HTML formatted labels using a custom IInputMode implementation and HtmlLabelStyle's findHyperlink method.

Things to Try

Move the mouse over one of the blue text segments representing hyperlinks. Note how the mouse cursor changes from the default cursor to a hand cursor while the mouse stays over the hyperlink. Additionally, a tool tip will display the corresponding hyper reference.

Clink one of the hyperlinks to activate it.
Headlines reference websites using the http protocol. Clicking on these links will either open the referenced website in a browser or (if the Java Desktop.Action#BROWSE is not supported) display the address in a dialog.
All other links use a custom graph protocol for referencing other nodes in the diagram. Clicking on these links will center the corresponding node in the diagram.
