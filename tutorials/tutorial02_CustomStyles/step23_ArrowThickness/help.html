Step 23: Arrow Thickness

Render the arrow depending on a property of the edge it belongs to. In this case, the thickness of the edge is considered when creating the path that defines the outline of the arrow.

Things to Try

  • Change the edge thickness in code and see how the arrow adapts to it.
