Step 12: Label Preferred Size

Adjust the size of the label depending on the size of its text by overriding the getPreferredSize() method.

Things to Try

  • Select a label, open the label editor using F2 to edit the label's text, and see the label size adjust.
  • Take a look at MySimpleLabelStyle.getPreferredSize().
