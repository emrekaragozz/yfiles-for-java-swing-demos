Description

This demo displays graphs in an isometric fashion to create an impression of a 3-dimensional view.

Interaction

As this demo is mainly a viewer, it is not possible to edit the graph structure.

  • Left Click on Group State Icon opens/closes the group node and shows/hides its children.
  • Left Drag on View moves view according to the difference between drag start and current mouse position.
  • Mouse Wheel changes the zoom level of the view.

Layout

There are two layout algorithms well-suited for arranging graphs in isometric views: Hierarchic layout and orthogonal layout.

By default, hierarchic layout is used.

However, the source code of the demo is organized in such a way that switching to orthogonal layout is a simple matter of changing the value of field layoutStyle in the main demo class.

Opening/closing group nodes automatically re-arranges the the current diagram with the chosen layout algorithm.

See the online Isometric Drawing Demo for additional interactive features.
